# Open Wind Data

Open Wind Data is an independent initiative to build 
the largest open repository of real-world wind turbine 
SCADA data for research and development.

## How it works

Contributors upload anonymized turbine datasets to our 
[Zenodo Community](https://zenodo.org/communities/openwinddata).
In return they gain access to all datasets in the repository.

## Who can contribute

- **Data contributors** – operators and service providers 
  who upload SCADA datasets
- **Research contributors** – researchers who publish 
  code or papers using the data

## Repository structure

```
schema/        → Signal schema and data format
templates/     → Upload templates
validation/    → Validation script
docs/          → Documentation
```
